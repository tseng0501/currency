# [虛擬貨幣記賬本](http://163.18.43.229:3800/src/index.html)
backEnd 資料夾是測試用的後端

frontEnd 資料夾是前端

 ----------

1. 首先安裝套件
```
npm install
```
2. 前往 frontEnd/config 裡的 config.json，修改proxy 的 ip 位置

3. 執行app.js檔案
```
npm start 或者 node app.js
```
